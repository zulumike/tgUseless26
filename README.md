# Use less

A simple web application that encourages users to use less mouse movement by implementing a custom cursor that moves faster within a designated area.

## Description

This project hides the default browser cursor and replaces it with a custom bike icon. When the cursor enters the red central area, it accelerates, promoting more efficient mouse usage.

## Features

- Custom cursor with bike icon
- Accelerated cursor movement in the central red zone
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your web browser.

## Files

- `index.html`: Main HTML structure
- `style.css`: Styling for the application
- `app.js`: JavaScript logic for cursor behavior
